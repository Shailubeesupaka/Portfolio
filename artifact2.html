Artifact Information

Introduction

This artifact presents an AI chatbot created to support intelligent and personalized travel planning.

It focuses on how prompt-driven AI can translate user intent into practical itinerary suggestions, making planning easier for people with different destinations, budgets, and travel goals.

Description

This artifact demonstrates the application of AI-assisted interaction design and prompt-based response generation in a user-centered planning scenario.

The artifact combines conversational interaction, travel context, and structured output design to show how AI can move from generic chat toward a usable planning assistant experience.

Objective

  • Explore AI chatbot design
  • Improve prompt-driven responses
  • Create a practical planning assistant

Process

  • Defined user travel scenarios
  • Designed prompts and response flows
  • Tested itinerary quality and clarity
  • Refined output for usability
  • Compared how different prompt wording changed itinerary quality
  • Adjusted the output structure so recommendations were easier to scan and apply

Tools and Technologies Used

Generative AI | Prompt Engineering | Chatbot Design

Value Proposition

This artifact shows how AI can be used to transform user intent into a practical and personalized planning experience.

Unique Value: Turns travel inputs into clear trip suggestions.

Relevance: Demonstrates applied AI thinking for user-focused solutions.

Reflection

This artifact strengthened my understanding of how prompt design and interface clarity shape the usefulness of AI-generated responses.
